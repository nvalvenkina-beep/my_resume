# Портфолио курс

Этот репозиторий содержит проект портфолио на Vite + React + TypeScript.

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Публикация на GitHub

1. Создайте репозиторий на GitHub.
2. Добавьте удалённый репозиторий:

```bash
git remote add origin https://github.com/<your-name>/<repo>.git
```

3. Отправьте изменения:

```bash
git branch -M main
git push -u origin main
```

4. Для GitHub Pages используйте ссылку:

https://nvalvenkina-beep.github.io/my_resume/

Если страница всё ещё показывает 404, проверьте в настройках репозитория GitHub → Pages → Source: Deploy from a branch и выберите ветку `gh-pages` или `main` с папкой `/root`.
